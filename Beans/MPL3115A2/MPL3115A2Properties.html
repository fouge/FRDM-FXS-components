Embedded Components
Component MPL3115A2
Driver for the Freescale Xtrinsic MPL3115A2 I2C Precision Altimeter.
Component Level: Low
Category: CPU External Devices-Sensor
Properties:
(Properties are parameters of the component. Please see the Embedded Components page for more information.)
  • Component Name - Name of the component.
  • Slave address -
    • 0x60: I2C slave address (Only one address accessible)

  • I2C bus - Link component (for details about settings see <a href="../../DOCs/InheritanceAndSharing.html">Component Inheritance &amp; Component Sharing</a>).
  • Barometer/Altimeter mode -

    There are 2 options:
    • Altimeter: Part is in Altimeter mode
    • Barometer: Part is in Barometer mode

  • Oversampling Ratio (OSR) - These bits select the oversampling ratio

    There are 8 options:
    • 1: Oversampling Ratio. Minimum time between data samples : 6ms
    • 2: Oversampling Ratio. Minimum time between data samples : 10ms
    • 4: Oversampling Ratio. Minimum time between data samples : 18ms
    • 8: Oversampling Ratio. Minimum time between data samples : 34ms
    • 16: Oversampling Ratio. Minimum time between data samples : 66ms
    • 32: Oversampling Ratio. Minimum time between data samples : 130ms
    • 64: Oversampling Ratio. Minimum time between data samples : 258ms
    • 128: Oversampling Ratio. Minimum time between data samples : 512ms

  • One shot mode - Get data using OST bit (FIFO disabled)
  • Time step - Auto Acquisition Time Step

    There are 7 options:
    • 1 second: Time step : 1s.
    • 2 seconds: Time step : 2s.
    • 4 seconds: Time step : 4s.
    • 8 seconds: Time step : 8s.
    • 16 seconds: Time step : 16s.
    • 32 seconds: Time step : 32s.
    • 64 seconds: Time step : 64s.
