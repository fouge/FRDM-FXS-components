Component FXAS21000
Driver for the Freescale FXAS21000 digital I2C gyroscope.
Component Level: Low
Category: CPU External Devices-Sensor
Properties:
(Properties are parameters of the component. Please see the Embedded Components page for more information.)
  • Component Name - Name of the component.
  • Slave Address -

    There are 2 options:
    • 0x20: Address 0x20, with SA0 (pin 13) set to 0.
    • 0x21: Address 0x21, with SA0 (pin 13) set to 1.

  • I2C bus - Interface to the I2C bus connected to the gyroscope
  • Output Data Rate - Select the output data rate

    There are 3 options:
    • 100Hz: Output Data Rate set to 100Hz (10ms)
    • 50Hz: Output Data Rate set to 50Hz (20ms)
    • 25Hz: Output Data Rate set to 25Hz (40ms)
